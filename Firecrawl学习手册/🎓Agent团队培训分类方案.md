## 🏆 培训考核体系

### 📝 考核方式（4 种）

#### 1. 理论考试（20%）

- **题型**: 选择题 + 简答题
- **内容**: Firecrawl API、最佳实践、成本优化
- **及格**: 70 分（满分 100）

#### 2. 代码作业（30%）

- **要求**: 提交可运行的 Python 脚本
- **评分标准**:
  - 功能完整性 (40%)
  - 代码质量 (30%)
  - 错误处理 (20%)
  - 文档注释 (10%)

#### 3. 实战项目（40%）

- **要求**: 完成指定的 HawaiiHub 案例
